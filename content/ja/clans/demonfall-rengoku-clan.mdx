<Section icon={Target} title="炎の呼吸の解放：ステップ・バイ・ステップ">

**Demonfall 煉獄家**の一員になったら、次の目標は炎の呼吸をマスターすることです。村で教わる他のスタイルとは異なり、炎の呼吸を習得するには危険な**ホワイトピーク（White Peak）**の頂上へ向かう必要があります。煉獄NPCに実力を証明するため、2つの困難な収集タスクを完了しなければなりません。

<Steps>
  <Step num="1" title="NPCの場所を特定">
    ホワイトピークへ向かいます。煉獄NPCは山の高い場所に位置しており、多くの場合、谷を見下ろす崖の近くに立っています。
  </Step>
  <Step num="2" title="鬼の角を集める">
    **鬼の角を20個**集める必要があります。これらはマップ中の一般的な鬼からドロップします。特に夜間や洞窟内が狙い目です。
  </Step>
  <Step num="3" title="鬼のボスを倒す">
    NPCから**鬼の首輪**を要求されます。ホワイトピークの洞窟にいる特定の鬼のボスを見つけて倒し、このアイテムを入手してください。
  </Step>
  <Step num="4" title="最終修行">
    アイテムを持ってNPCのもとへ戻ります。炎の呼吸のスタイルが授けられ、スキルツリーの最初の5つの型が解放されます。
  </Step>
</Steps>

<Callout type="warning" title="環境の危険性">
ホワイトピークは極寒で、高レベルの鬼が徘徊しています。登山を試みる前に、体力回復用のスープを十分に用意しておきましょう。
</Callout>

| 必要条件 | 数量 | 入手先 |
| :--- | :--- | :--- |
| **鬼の角** | 20 | 一般的な鬼のドロップ |
| **鬼の首輪** | 1 | ホワイトピーク洞窟のボス |
| **スキルポイント** | 変動 | 戦闘によるレベルアップ |
| **円** | 0 | アイテム提供後の修行自体は無料 |

</Section>

<Section icon={Swords} title="炎の呼吸の技セットと玖ノ型">

**Demonfall 煉獄家**の真の力は、拡張された技セットにあります。通常の炎の呼吸の使い手は伍ノ型までしか使えませんが、煉獄家の一員は「玖ノ型：煉獄」を解放できます。この技は超高速の突進とともに巨大な炎の龍を生成し、進路上にいる全員に多段ヒットダメージを与えます。

| 型の番号 | 名称 | 説明 |
| :--- | :--- | :--- |
| **壱ノ型** | 不知火 | 高速の突進後、一撃で斬りつける。 |
| **弐ノ型** | 昇り炎天 | 敵を打ち上げるのに適した、上方向への弧を描く斬撃。 |
| **参ノ型** | 気炎万象 | 飛来物をそらす円形の斬撃。 |
| **肆ノ型** | 盛炎のうねり | 攻撃をパリィする防御的なカウンター技。 |
| **伍ノ型** | 炎虎 | 虎の形を模した連続斬撃。 |
| **玖ノ型** | **煉獄** | **一族限定**: 広範囲に壊滅的なダメージを与える突進技。 |

<Callout type="success" title="無限呼吸">
これらの技を最大限に活用するために、鎌狩村（Kamakari Village）へ行き、300円で**岩（Boulder）**を購入しましょう。これを使って、呼吸を止めている間スタミナバーが減らなくなるまで呼吸を訓練してください。これにより、型を絶え間なく連発できるようになります。
</Callout>

</Section>

<Section icon={Shield} title="炎の日輪刀の鍛造">

炎の柱としての変貌を完成させるには、カスタムの**炎の日輪刀**を鍛造する必要があります。これには鬼殺隊本部へ行き、刀鍛冶と話す必要があります。デフォルトの色はランダムなので、象徴的な赤い刀身を得るためにはリロールが必要になる場合があります。

| アイテム | コスト | 必要条件 |
| :--- | :--- | :--- |
| **隊士の日輪刀** | 無料 | 最終選別を完了する |
| **炎の日輪刀** | 10,000円 | クリスタルエッセンス 1個 + 太陽鉱石 20個 |
| **色のリロール** | 3,000円 | 赤/橙が出るかはランダム |
| **煉獄の鍔** | 特殊 | 煉獄家の一員であること |

<Callout type="tip" title="刀身の色">
もしRobuxに余裕があれば、刀身の色のリロールは**50 Robux**で行えます。煉獄ビルドには「赤」が最も正確ですが、「橙」も炎のエフェクトによく映えます。
</Callout>

</Section>
