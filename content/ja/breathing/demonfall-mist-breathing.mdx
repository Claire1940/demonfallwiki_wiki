<Section icon={Map} title="Demonfall 霞の呼吸の必要条件と円稼ぎ">

霞の使い手になるための旅に出る前に、キャラクターが参加条件を満たしているか確認する必要があります。一部の基本スタイルとは異なり、霞の呼吸には2,000円という多額の投資が必要です。この通貨は主に、ゲーム内の各村で見つかる装飾品（Trinkets）を売ることで入手できます。

**動画の見どころ：**
- **初期費用**：トレーナーと話すには最低2,000円が必要です。
- **装飾品集めのルート**：早川村の机、ベッド、狭い路地をチェックするのが効率的な稼ぎ方です。
- **トレーナーへの道**：村のスポーン地点から出発し、特定の木を通り過ぎて雪の断崖に向かいます。

<YouTubeEmbed videoId="VJkyl_0gHLM" title="霞の呼吸 完全ガイド + 場所 + 迷路攻略 | Roblox Demonfall" />

2,000円を素早く貯めるには、プレイヤーは「装飾品ラン（Trinket Run）」に集中すべきです。装飾品は指輪やカフスなどの小さなアイテムで、決まった場所に出現します。以下の表は、探索中に見つかる最も一般的なアイテムをまとめたものです。

| アイテムの種類 | 推定価値 | 主な出現場所 |
| :--- | :--- | :--- |
| **壊れた指輪** | 少額 | 机の上、ベッドの下 |
| **古いカフス** | 中額 | 村の壁の裏、路地の隅 |
| **金の指輪** | 高額 | 建物の内部（レア出現） |
| **鬼の角** | 変動 | 郊外の雑魚鬼がドロップ |

<Callout type="tip" title="円稼ぎの効率">
早川村の部屋から始めるのがベストなルートです。すべての棚や机をチェックしましょう。その後、外壁の裏に飛び、隅々を確認してください。装飾品は主要な建物の裏側に固まって出現することが多いです。
</Callout>

<Section icon={Navigation} title="トレーナー時透無一郎の見つけ方">

時透無一郎を見つけることが最初の大きな難関です。彼は一般的な村にはおらず、雪深い山岳地帯に隠された人里離れたキャンプに住んでいます。迷わないためには正確なナビゲーションが必要です。

<CardGrid cols={3}>
  <Card title="出発地点">
    **早川村**の出口から始めます。大きな登れる壁に着くまで灰色の石の道を進んでください。
  </Card>
  <Card title="一本の木">
    一本だけ立っている木を探してください。前進する際は、この木の**右側**を通るようにします。
  </Card>
  <Card title="雪の断崖">
    崖の縁へ向かいます。飛び降りるのではなく、中央に雪が流れている岩の方へ**北西**を向いて進みます。
  </Card>
</CardGrid>

特徴的な雪の模様がある岩に着いたら、その左側を回り込みます。すると無一郎のいるキャンプがすぐに見えます。しかし、たどり着くだけでは不十分です。実力を証明するために「霞の迷路」を突破しなければなりません。

| 目印 | 方向 | 識別方法 |
| :--- | :--- | :--- |
| **主要な道** | 右 | スポーン地点からの灰色の石の道を辿る |
| **石壁** | 上 | 下の谷を避けるためにここを登る |
| **枯れ木** | 右 | この目印の右側を維持する |
| **霜の岩** | 左 | キャンプを見るために左側を回り込む |

<Callout type="warning" title="準備に関する警告">
キャンプに入る前に2,000円を用意しておきましょう。資金がない状態で無一郎に話しかけても試練を開始できず、移動時間が無駄になってしまいます。
</Callout>

<Section icon={Target} title="霞の迷路の攻略手順">

霞の迷路は、その紛らわしいレイアウトと見えない障壁で悪名高いです。道が視覚的に直感的ではないため、多くのプレイヤーがここで苦戦します。霞の呼吸を授かる中心部に到達するには、特定の順序で曲がるしかありません。

<Steps>
  <Step num="1" title="最初の進入">
    迷路に入ったらすぐに**左**に曲がり、最初の障壁にぶつかるまで直進します。
  </Step>
  <Step num="2" title="中間セクション">
    **右**、次に**左**、そしてもう一度**右**に曲がります。この順序で最初の行き止まりを回避できます。
  </Step>
  <Step num="3" title="狭い通路">
    素早く**左、右、左、右**と曲がり続けます。やがて狭い道にたどり着きます。
  </Step>
  <Step num="4" title="最後の直線">
    一番長い廊下を突き当たりまで進み、最初の横の開口部は無視して、左側にある**最も細い通路**に入るとゴールです。
  </Step>
</Steps>

| 移動ステップ | 方向 | アクション |
| :--- | :--- | :--- |
| **ステップ 1-3** | 左 -> 直進 -> 左 | 迷路の核への初期進入 |
| **ステップ 4-6** | 右 -> 直進 -> 右 | 中間バイパスの通過 |
| **ステップ 7-10** | 左 -> 右 -> 左 -> 右 | ジグザグセクションへの進入 |
| **ステップ 11-13** | 下 -> 左 -> 狭い道 | トレーナーのプラットフォームに到達 |

<Callout type="success" title="迷路のヒント">
もし行き詰まったら、右側の壁に沿って進んでみてください。ただし、上記の順序が無一郎への最短ルートです。彼にたどり着いたら、話しかけて修行を完了させましょう。
</Callout>

</Section>

<Section icon={Swords} title="霞の呼吸の技セットと戦闘戦略">

Demonfallの霞の呼吸は、相手を混乱させる能力が特徴です。「視界を遮る」メカニズムを利用することで、敵が次の動きを予測するのを困難にします。これは特にPvP（対人戦）において非常に強力です。

**戦闘の基本メカニズム：**
- **酸素ゲージ**：すべての呼吸スタイルと同様に、霞も酸素を必要とします。**G**キーを長押しして呼吸し、ゲージを回復させます。
- **機動力重視**：霞の技にはダッシュやテレポートが含まれることが多く、距離を詰めたりコンボから逃れたりすることができます。
- **視覚的な撹乱**：いくつかの技は、キャラクターのアニメーションを隠す霞の雲を作り出します。

| 技名 | タイプ | 戦闘での有用性 |
| :--- | :--- | :--- |
| **朧 (Obscuring Clouds)** | ユーティリティ/ダッシュ | 雲を作り出し、素早い位置取りを可能にする |
| **八重霞 (Eight Layered Mist)** | 多段ヒット | 狭い範囲での連続した高速斬撃 |
| **遠霞 (Distant Horizon)** | 範囲/AOE | 離れた敵を攻撃する広範囲の薙ぎ払い |
| **霞雲の海 (Lunar Dispersing Mist)** | 防御 | 攻撃的なプレイヤーを返り討ちにするカウンター技 |

霞の呼吸は主に**筋力（Strength）**と**呼吸（Breathing）**のステータスに応じて強化されます。ダメージを最大化するには、呼吸ゲージの容量をアップグレードすることに集中し、リチャージのために中断することなく長いコンボを繰り出せるようにしましょう。

<Callout type="info" title="PvPの戦略">
PvPでは、「朧」を使って相手のダッシュを誘い出しましょう。相手のスタミナが切れたら、「八重霞」を叩き込んで最大ダメージを与えます。
</Callout>

</Section>
