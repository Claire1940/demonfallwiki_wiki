<Section icon={Map} title="最初の目覚めと鬼殺隊への道">

**Demonfallのストーリーライン**を巡る旅は、家族が襲撃される悲痛なシネマティック演出から始まります。これは、人間としての道を進むか、あるいは最終的に鬼へと堕ちるか、あなたのキャラクターが成長する契機となります。鬼殺隊の道を目指す者にとって、最初の鬼との遭遇は、戦闘技術の最初の試練となります。

**動画の見どころ：**
- 序盤の任務における、縁壱スペックの圧倒的な速度とパワーの実演。
- 最初の家族の家と最初の鬼との遭遇の攻略。
- 旅のリセットや開始を行うための、ミランダ（Miranda）などの重要NPCとの対話。
- 修行フェーズに素早く到達するための、序盤の会話シーケンスの効率的なスキップ。

<Section icon={Target} title="鬼殺隊試験と最終選別">

**Demonfallのストーリーライン**を進行させるには、すべての鬼殺隊志望者が最終選別を突破しなければなりません。この過酷な試験は、鬼がはびこる森で行われ、生き残ることだけが目的となります。鬼殺隊に実力を証明するために、規定の数の鬼を討伐する必要があります。

<Steps>
  <Step num="1" title="鱗滝を見つける">
    鱗滝（Akira Urokodaki）を見つけて、最終選別のクエストを開始します。彼は必要な指導と、試験に不可欠な象徴的な厄除の面を授けてくれます。
  </Step>
  <Step num="2" title="森に入る">
    試験場へ移動します。一度中に入ると、目的を達成するか敗北するまで脱出することはできません。
  </Step>
  <Step num="3" title="鬼の討伐">
    合計で30体の鬼を倒す必要があります。これには通常の鬼だけでなく、より高いHPと強力な攻撃を持つ、危険な「青い鬼」も含まれます。
  </Step>
  <Step num="4" title="完了">
    30体の討伐を達成した後、出口に戻ることで鬼殺隊の隊服と、組織内での最初の公式階級を受け取ることができます。
  </Step>
</Steps>

| 鬼の種類 | 難易度 | 対策・戦略 |
| :--- | :--- | :--- |
| **通常の鬼** | 低 | 基本的なM1コンボと重攻撃 |
| **青い鬼** | 中 | 呼吸の型を使用してガードを崩す |
| **集団モブ** | 高 | 「陽華突（Clear Blue Sky）」などの範囲（AoE）スキルが不可欠 |

<Callout type="warning" title="試験の要件">
30体の討伐しきい値に達する前に森を出ようとしないでください。ゲームは進行状況を記録しており、途中で退出すると不合格となり、試験を最初からやり直すことになります。
</Callout>

</Section>

<Section icon={Swords} title="下弦の鬼との対決">

鬼殺隊に入隊すると、**Demonfallのストーリーライン**は、産屋敷耀哉（Kagaya Ubuyashiki）から下される命がけの任務へとシフトします。これらの任務には、罪のない民間人の防衛や十二鬼月の追跡が含まれます。独自の血鬼術を持つボスたちと遭遇するため、難易度は劇的に上昇します。

<CardGrid cols={3}>
  <Card title="亜久呂（下弦の陸）">
    - **場所**: 最終選別の後続クエスト
    - **難易度**: 普通
    - **重要なコツ**: 足への攻撃に集中して機動力を制限する
  </Card>
  <Card title="累（下弦の伍）">
    - **場所**: 那田蜘蛛山
    - **難易度**: 高

  </Card>
  <Card title="下弦の弐">
    - **場所**: 最終ストーリーチャプター
    - **難易度**: 極限
    - **重要なコツ**: 完璧な受け流し（パリィ）とスタミナ管理が必須
  </Card>
</CardGrid>

亜久呂との戦闘はボス戦のメカニズムを学ぶ入門編であり、予兆のある攻撃を回避し、隙を突くことが求められます。一方、累は遠距離からの糸攻撃を繰り出し、プレイヤーを拘束して一気にHPを削ってきます。下弦 of 弐との遭遇は、現在のストーリーライン進行における最高峰の試練であり、選択した「呼吸の型」について学んだすべての技術が試されます。

| ボス名 | HP量 | 主なドロップ |
| :--- | :--- | :--- |
| **亜久呂** | 約1,500 HP | 経験値、円（Yen）、隊士ランクアップ |
| **累** | 約2,500 HP | 大量の経験値、希少な素材 |
| **下弦の弐** | 約4,000 HP | 最高峰の経験値、ストーリー完了 |

<Callout type="success" title="戦闘効率">
「烈日紅鏡（Solar Heat Haze）」や「陽華突（Clear Blue Sky）」などの日の呼吸の技を使用すると、ボス戦で最高のダメージ対スタミナ比を得られます。究極技からダッシュで逃れられるよう、常に少なくとも20%のスタミナを維持してください。
</Callout>

</Section>

<Section icon={Shield} title="村の防衛と鬼殺隊の任務">

主要なボス戦の合間に、**Demonfallのストーリーライン**では様々な鬼殺隊の任務をクリアする必要があります。これらの任務では、襲撃を受けている村へ赴くことが多くあります。産屋敷耀哉があなたを本部に召集し、これらの緊急事態についての説明を行います。

村の防衛任務は耐久力の試練です。鬼はウェーブ形式で出現するため、NPCの「生存している村人」の数が減りすぎないよう、効率的に処理する必要があります。この任務は、累のボス戦を解放し、さらに後半のコンテンツへと進むための必須条件となっています。

| 任務フェーズ | 敵の数 | 完了条件 |
| :--- | :--- | :--- |
| **到着** | 鬼 5体 | 入口の門を制圧する |
| **内部掃討** | 鬼 10体 | 村の中心部を守る |
| **最終ウェーブ** | エリート鬼 5体 | マークされたすべての標的を倒す |

<Callout type="info" title="NPCとの対話">
耀哉があなたを「我が子」と呼ぶとき、それは鬼殺隊内でのあなたの地位向上を意味しています。これらの会話シーケンスは、より高難易度の任務ティアを解放するために不可欠です。
</Callout>

</Section>

<Section icon={Sparkles} title="高度な進行とスペック">

現在の**Demonfallストーリーライン**の終盤に達すると、プレイヤーは自身のビルドの洗練に集中できるようになります。幸運にも「縁壱スペック（Yoriichi Spec）」を手に入れたプレイヤーにとって、ゲームプレイは生存のための死闘から、圧倒的な力で無双するファンタジーへと変化します。このスペックが提供するステータスは、2026年のメタにおいて他の追随を許しません。

| 属性・ステータス | 通常の隊士 | 縁壱スペック |
| :--- | :--- | :--- |
| **最大HP** | 100 - 300 | 550 |
| **最大スタミナ** | 100 - 200 | 500 |
| **歩行速度** | 標準 | 強化（超高速） |
| **固有スキル** | なし | 日の呼吸の極意 |

この強さを維持するために、プレイヤーは経験値目的で下弦のボスたちを継続的に周回する必要があります。メインストーリーをクリアした後でも、「下弦の弐」との戦闘はレベル上限に達し、上弦の鬼が登場する将来のコンテンツアップデートに備えるための最も効率的な方法であり続けます。

<Callout type="tip" title="周回ルート">
ストーリーラインが終了した後の最適な経験値周回ルートは、下弦の弐 → 累 → 村の防衛 です。このサイクルにより、時間あたりの獲得経験値を最大化しつつ、リロール用の円（Yen）を大量に蓄えることができます。
</Callout>

</Section>
