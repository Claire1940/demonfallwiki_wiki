<Section icon={Map} title="コントローラーのレイアウトとセットアップの概要">

Demonfall はコンソールとコントローラーのサポートを大幅に改善し、PC プレイヤーとコンソール プレイヤーの差を縮めました。以前は、マウスとキーボードなしではメニューの操作、壁登りの実行、戦闘時のカメラアングルの管理が非常に困難でした。専用のコントローラーマッピングの統合により、ゲームプレイ体験は完全に一新され、コンソール プレイヤーも最高レベルで競い合うことが可能になりました。

**ビデオのハイライト:**
- コンソールコントローラーでシフトロックをシームレスに切り替える方法。
- メニュー操作と、インベントリアイテムをアクティブスロットに装備する方法。
- 戦闘、担ぐ、とどめを刺すための正確なボタンの組み合わせ。
- コンソールでの一般的な NPC インタラクションのバグを解決する方法。

<YouTubeEmbed videoId="Sj-T7d8cKSs" title="Demonfall コントローラーキーバインドのアップデート" />

コントローラーユーザーから最も要望の多かった機能の一つが、専用のシフトロック切り替えでした。このメカニクスは現在完全に機能しており、プレイヤーはカメラの視点を固定できます。これは、正確な呼吸の技を当てたり、複雑なパルクール操作を実行したりする際に非常に重要です。

| アクション | コントローラー入力 | キーボード入力 |
| --- | --- | --- |
| シフトロック切り替え | LB + RB | Shift |
| メニュー / プレイヤーを開く | 方向キー下 | Tab / M |
| スキルツリーを開く | 方向キー右 | K |
| 呼吸を溜める | 方向キー上 | G |
| NPC とインタラクト | 方向キー左 | E |

<Callout type="warning" title="NPC インタラクションのバグ">
一部の NPC は配置のバグにより、コントローラーで話しかけるのが難しい場合があります。インタラクトのプロンプトが表示されない場合は、NPC に極限まで近づくか、カメラを NPC の頭の真上に移動させてから方向キー左を押してみてください。
</Callout>

<Section icon={Swords} title="戦闘メカニクスとコントローラーのマッピング">

Demonfall の戦闘は展開が早く、非常にシビアであり、タイミングとポジショニングが重視されます。基本的な弱攻撃コンボ（M1）を実行する場合でも、壊滅的な強攻撃を繰り出す場合でも、コントローラーやキーボードのレイアウトを熟知しておくことが不可欠です。アクション性の高いゲームであるため、一度の入力ミスが体力の大部分を失う結果につながることもあります。

コントローラープレイヤーの場合、レイアウトは基本攻撃と移動にフェイスボタンを使用し、アクティブスキルの切り替えにはトリガーを使用します。キーボードプレイヤーは、標準的なマウスクリックとホットキーを頼りに、呼吸の型や血鬼術（BDA）を使い分けます。

| アクション | Xbox 入力 | PlayStation 入力 |
| --- | --- | --- |
| 弱攻撃 (M1) | X ボタン | □ ボタン |
| 強攻撃 (M2) | Y ボタン | △ ボタン |
| ダッシュ | B ボタン | 〇 ボタン |
| ジャンプ | A ボタン | ✕ ボタン |
| 相手を担ぐ | RB + B | R1 + 〇 |
| 相手にとどめを刺す | RB + Y | R1 + △ |

| アクション | デフォルト PC キー | 戦闘機能 |
| --- | --- | --- |
| 弱攻撃 (M1) | 左クリック | コンボの起点およびプレッシャー用ツール |
| 強攻撃 (M2) | 右クリック | ガード崩しおよび高ダメージのフィニッシャー |
| ダッシュ | Q キー | 位置取りやコンボ脱出のための素早い回避 |
| ブロック | F キー | 前方からの攻撃によるダメージを軽減 |
| 相手を担ぐ | V キー | ダウンしたプレイヤーや NPC を担ぎ上げる |
| 相手にとどめを刺す | B キー | ダウンした標的にとどめを刺し、経験値や体力を得る |

<Callout type="tip" title="パーフェクトブロックカウンター">
相手の攻撃が当たる直前にブロックすることで、パーフェクトブロック（PB）を実行できます。PB に成功すると自動的にカウンターキックが発動して相手を吹き飛ばし、即座に反撃のチャンスを作ることができます。
</Callout>

<Section icon={Sparkles} title="高度な移動とパルクールチュートリアル">

垂直方向の機動力は、Demonfall の世界探索における核心的な要素です。呼吸の指導者、隠れた NPC、またはクエストの目的地に到達するために、高い崖を登ったり、屋根を移動したり、山脈を越えたりする必要があります。標準のダブルジャンプだけでは高い垂直の壁を登るには不十分なため、プレイヤーは壁登りチェインのメカニクスをマスターしなければなりません。

パルクール中はシフトロックの使用を強く推奨します。カメラを固定することで、垂直な面の間を飛び跳ねる際のキャラクターの向きを制御しやすくなります。

<Steps>
  <Step num="1" title="シフトロックを有効にする">
    コントローラーでは LB と RB を同時に押し、キーボードでは Shift キーを押して、カメラ視点を固定します。これにより、キャラクターは常にカメラの方向を向くようになります。
  </Step>
  <Step num="2" title="壁に向かってジャンプする">
    垂直な面に向かって走り、ジャンプボタン（Xbox は A、PlayStation は X、PC はスペースキー）を押して、壁に最初に接触します。
  </Step>
  <Step num="3" title="回転して再度ジャンプする">
    壁に触れている間に、素早くカメラを回転させて反対側の壁や角度を向き、再びジャンプボタンを押して上方へ跳ね返ります。
  </Step>
  <Step num="4" title="ジャンプを繋げて頂上へ">
    カメラの回転とジャンプのシーケンスを素早く繰り返します。これにより、ずり落ちることなく非常に高い壁を登ることができます。
  </Step>
</Steps>

<Callout type="info" title="移動スタミナ">
壁登りは呼吸ゲージを消費しませんが、ダッシュは消費します。登っている途中で落下しないよう、長距離のパルクールを行う前にはスタミナバーが全快していることを確認してください。
</Callout>

</Section>

<Section icon={Shield} title="サバイバルとインベントリ管理">

Demonfall での生存には、ステータスバーの慎重な管理が必要です。体力とスタミナに加えて、プレイヤーは空腹ゲージを注意深く監視しなければなりません。空腹ゲージが空になると、キャラクターの自然回復が完全に停止し、あらゆる攻撃に対して非常に脆弱になります。

インベントリを効率的に管理することで、激しい戦闘中や戦闘後に素早く回復できるようになります。

<CardGrid cols={3}>
  <Card title="隊士の生存">
    隊士は村の商人からスープを購入することに大きく依存しています。スープを飲むことで空腹ゲージが回復し、長時間の PvE 周回中も継続的な体力回復が可能になります。
  </Card>
  <Card title="鬼の生存">
    鬼は空腹を満たすために、倒した人間の NPC やプレイヤーを喰らわなければなりません。鬼の空腹ゲージが空になると体力を回復できなくなり、戦闘で瞬時に敗北する原因となります。
  </Card>
  <Card title="インベントリのホットバー">
    アイテムはバッグメニューから Z および X スロットに装備する必要があります。トリガー（RT/LT）を使用してこれらのスロットを選択し、インタラクトボタンを押すことで、戦闘中にスープなどのアイテムを摂取できます。
  </Card>
</CardGrid>

| 指標 | 画面上の位置 | 枯渇時のペナルティ | 主な回復方法 |
| --- | --- | --- | --- |
| 体力バー | 下部中央 | キャラクターが死亡し経験値を失う | 自然回復（空腹状態が必要） |
| スタミナバー | 左下（青） | ダッシュやスキルの発動が不可 | 時間経過による自動回復 |
| 空腹バー | 右下 | すべての体力回復が停止 | スープ、肉の摂取、または捕食 |

<Callout type="success" title="スープの効率">
インベントリには常に最大数のスープを持ち歩きましょう。これは空腹ゲージを維持するための最も費用対効果が高く信頼できる方法であり、予期せぬ PvP 遭遇に常に備えることができます。
</Callout>

</Section>

<Section icon={Trophy} title="スキルツリーと進行の節目">

キャラクターの強さは、レベルとスキルツリーへのポイント割り当てによって決まります。コントローラーでスキルツリーにアクセスするには方向キー右を押します。メニューは呼吸の技、身体強化、剣術に分かれています。鬼の場合、呼吸のセクションは血鬼術（BDA）に置き換わります。

一度消費したスキルポイントは外すことができないため、事前にビルドを計画することが重要です。スキルをリセットする唯一の方法は、ワイプポーション（Wipe Potion）を使用するか、キャラクターをプレステージさせることです。
