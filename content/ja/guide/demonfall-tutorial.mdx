<Section icon={Sparkles} title="Demonfallチュートリアル：キャラクター作成と一族ティアリスト">

ゲームを最初に起動すると、キャラクターカスタマイズ画面が表示されます。ここでは、見た目のカスタマイズと家族（一族）のロール（ガチャ）を行うことができます。家族の一族によってパッシブ能力が決まり、進行速度、戦闘効率、全体的なプレイスタイルに大きく影響します。

**ビデオのハイライト:**
- **Trelloガイド**: 一族のパッシブやマップの場所を確認するために、公式Trelloを開いておきましょう。
- **一族の選択**: レベル上げ速度を最大化するため、可能であれば竈門（Kamado）を狙いましょう。
- **最終選別の準備**: 試験に向かう前に、体力回復用のパンを購入しておきましょう。
- **瓢箪訓練**: 隠れ里（Okuya Village）で瓢箪を購入し、呼吸レベルを0から10まで上げましょう。

<Section icon={Swords} title="進む道の選択：鬼殺隊 vs 鬼">

キャラクターカスタマイズの直後、鬼とのチュートリアル戦闘に投げ出されます。この戦いの結果によって、あなたの種族とゲームプレイの軌道が決まります。

錆びた刀を使って鬼を倒すと、人間のまま**鬼殺隊（Slayer）**ルートに進みます。この戦いに勝つには、基本的な弱攻撃（M1）コンボと、敵がガードしている時にガードを崩す強攻撃（M2）を使いましょう。苦戦する場合は、ダッシュフリップで距離を取り、隙を待ちましょう。

鬼に敗北すると、キャラクターは**鬼（Demon）**に変貌します。このルートでは「血鬼術」と呼ばれる独自の超能力を得られますが、太陽光や日輪刀に対して脆弱になります。

| 項目 | 鬼殺隊ルート | 鬼ルート |
| :--- | :--- | :--- |
| **主要武器** | 日輪刀（試験後に作成） | 拳、爪、血鬼術 |
| **力の源** | 呼吸法（風、炎、雷など） | 血鬼術 ＆ 鬼スキル |
| **環境リスク** | 通常の落下ダメージと敵の攻撃 | 太陽光に当たると深刻なダメージ |
| **進行目標** | 最終選別を突破し、鬼殺隊に入隊 | 人間の魂を喰らい、鬼殺隊を倒す |
| **育成難易度** | 普通（クエストと瓢箪が必要） | 序盤は難しいが、PvPポテンシャルが高い |

<Callout type="info" title="鬼殺隊の人気">
象徴的な呼吸法と構造化された進行ルートがあるため、プレイヤーの約80%が鬼殺隊ルートを選択します。初心者の場合は、学習曲線が緩やかな鬼殺隊から始めることを強くお勧めします。
</Callout>

</Section>

<Section icon={Target} title="最終選別試験の突破">

鬼殺隊の正式な隊員になるには、藤襲山（Sakura Passage）へ向かい、最終選別試験を完了する必要があります。この試験では、鬼が蔓延る森で生き残り、30体の鬼を倒さなければなりません。

<Steps>
  <Step num="1" title="持ち物の準備">
    試験に向かう前に、隠れ里（Okuya Village）の店を訪れましょう。パンをたくさん購入してください。空腹ゲージはすぐに減り、長い試験中にパッシブで体力を回復させるために食べ物は不可欠です。
  </Step>
  <Step num="2" title="鬼の角クエストの受注">
    隠れ里のすぐ外にいる風の呼吸のトレーナーに話しかけましょう。彼は鬼の角を20個集めるよう依頼してきます。試験前にこのクエストを受けておけば、試験中に倒す30体の鬼から自動的に必要な角がドロップします。
  </Step>
  <Step num="3" title="藤襲山への移動">
    隠れ里の右側の道を進み、「Sakura Passage」の看板が見えるまで進みます。このエリアを通り抜けると、最終選別の門に到着します。
  </Step>
  <Step num="4" title="安全に30体の鬼を撃破">
    中に入ったら、一度に1〜2体の通常の鬼だけを引き寄せるように集中しましょう。基本的なM1コンボを使います。鬼がガードしたら、ダッシュフリップでリセットするか、右クリックの強攻撃でガードを崩しましょう。
  </Step>
  <Step num="5" title="日輪刀の鋳造">
    試験を生き延びたら、鍛冶屋のNPCに話しかけ、報酬として受け取った太陽の鉱石を使って自分専用の日輪刀を作成しましょう。刀の色はランダムですが、後でリセマラ可能です。
  </Step>
</Steps>

| 敵のタイプ | 危険度 | 主な攻撃 | 対策 |
| :--- | :--- | :--- | :--- |
| **通常の鬼** | 低 | 基本的な爪攻撃、ガード多用 | M1連打、右クリックでガード崩し、フリップで離脱 |
| **影の鬼** | 中 | 速いダッシュ攻撃、素早いコンボ | 最初のダッシュをジャストガードし、反撃する |
| **青い鬼** | 高 (ボス) | 地面踏みつけ、火/岩の柱攻撃 | **完全に避ける**。ターゲットにされたら逃げる |

<Callout type="warning" title="青い鬼とは戦わないこと">
青い鬼は高レベルプレイヤー向けのオプションボスです。初心者にとっては時間の無駄であり、倒しても特別な報酬はありません。絶対に避けてください。
</Callout>

</Section>

<Section icon={Shield} title="呼吸法と瓢箪訓練">

呼吸法は鬼殺隊の戦闘システムの核です。呼吸スキルを解放し効果的に使うには、瓢箪（ひょうたん）を使って肺を鍛える必要があります。このプロセスは「瓢箪訓練」と呼ばれます。

瓢箪は隠れ里の商人から1つ500円で購入できます。訓練するには、瓢箪を装備してマウスをクリックして息を吹き込みます。ミニゲームが表示されるので、プログレスバーを中央の安全圏に保つ必要があります。バーが赤いゾーンに入ってしまうと瓢箪が割れ、投資が無駄になります。

呼吸レベルは0から始まり、最大レベル10まで上がります。瓢箪を割るごとに呼吸レベルが上がり、息切れすることなくより強力なスキルを繰り出せるようになります。

<CardGrid cols={3}>
  <Card title="風の呼吸">
    - **トレーナーの場所**: 隠れ里（Okuya Village）
    - **難易度**: 非常に簡単（角20個 + ボス戦）
    - **スタイル**: 高い機動力、クラウドコントロール、スーパーアーマー技
  </Card>
  <Card title="雷の呼吸">
    - **トレーナーの場所**: 丘の上の家
    - **難易度**: 普通（4,000円が必要）
    - **スタイル**: 圧倒的なスピード、高速ダッシュコンボ、高い瞬間火力
  </Card>
  <Card title="炎の呼吸">
    - **トレーナーの場所**: 炎のトレーナーの山
    - **難易度**: 難しい（複雑なクエストラインが必要）
    - **スタイル**: 強力な燃焼ダメージ、広範囲への衝撃攻撃
  </Card>
</CardGrid>

| 呼吸レベル | 必要な瓢箪のタイプ | 1つあたりの価格 | 主なメリット |
| :--- | :--- | :--- | :--- |
| **レベル 0 〜 3** | 小さな瓢箪 | 500円 | 基本的な呼吸スキルの解放 |
| **レベル 4 〜 7** | 中くらいの瓢箪 | 1,200円 | 呼吸ゲージの最大容量アップ |
| **レベル 8 〜 10** | 大きな瓢箪 | 3,000円 | 呼吸の回復速度を最大化 |

<Callout type="success" title="風の呼吸の利点">
初心者には風の呼吸が強く推奨されます。クエストはトレーナーとの単純な決闘だけで、右クリックのガード崩しを使えば簡単にハメることができます。
</Callout>

</Section>

<Section icon={Trophy} title="序盤のレベル上げと進行ルート">

呼吸法と日輪刀を手に入れたら、主な目標はレベルを上げ、円を稼ぎ、スキルツリーのアクティブスキルを解放することです。

序盤に素早く円を稼ぐには、村の地面を探索して貴重な装飾品（Trinkets）を見つけ、商人に売りましょう。また、つるはしを購入して、山岳地帯に点在する鉱脈から鉱石を採掘することもできます。

経験値については、設定メニューで「鎹鴉（Crow）」をオンにしてください。鴉は定期的に討伐や配達のクエストを割り当て、これらはかなりの経験値と円をもたらします。一部のプレイヤーは早い段階で「奥呂ダンジョン（下弦の陸）」に挑戦しようとしますが、バグが多く、ソロの初心者にはお勧めしません。
