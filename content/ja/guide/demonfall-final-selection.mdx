<Section icon={Target} title="最終選別試練の理解">

**Demonfallの最終選別（demonfall final selection）**は、Demonfallの世界で戦士を目指すすべてのプレイヤーにとって最も重要なマイルストーンです。専用の挑戦エリアに位置するこの試練は、単なる人間から鬼殺隊の正式な隊員へと成長するための登竜門となります。藤の花に囲まれた鬱蒼とした森の中で、プレイヤーは様々な鬼を相手に自身の戦闘能力を証明しなければなりません。

現在の2026年の環境（メタ）においても、この試練は進行における重要な関門であり続けています。失敗した場合は浪人（Ronin）にとどまるか、キャラクターをリセットして別の道を模索する必要があります。しかし、合格すれば、より高ランクのボスを狩ったり、エンドゲームのレイドに参加したりするために必要なツールが手に入ります。森の中は暗く危険に満ちており、体力だけでなく、スタミナと空腹度も同時に管理する必要があります。

<Section icon={Shield} title="準備と必要条件">

**Demonfallの最終選別**を突破するための戦いは、森に入るずっと前から始まっています。多くのプレイヤーは、アイテムを持たずにレベル5で急いで挑戦し、圧倒的な数の鬼に囲まれて挫折してしまいます。適切な準備として、**早川村（Hayakawa Village）**や**オクヤ村（Okuyia Village）**を訪れて物資を集め、有利な一族（Clan）のバフを厳選（ロール）しておくことが推奨されます。

どの一族に所属するかによって、選別の難易度は劇的に変化します。例えば、**竈門（Kamado）**一族は膨大なスタミナ強化を提供し、**時透（Tokito）**一族はより早い経験値獲得を可能にします。たとえレジェンダリーの一族でなくても、初期装備の木刀を使って、基本的なM1（弱攻撃）とM2（強攻撃）のコンボをマスターすることに集中しましょう。

| 準備カテゴリ | 推奨アイテム/ステータス |
| :--- | :--- |
| **消耗品** | スープ 5個、包帯 3個 |
| **ステータス配分** | 腕力（Strength）に2ポイント、体力（Vitality）に2ポイント |
| **戦闘スキル** | 「ガードカウンター（Block-Counter）」技術の習得 |
| **一族の優先度** | 体力やスタミナのバフを持つ一族 |
| **資金** | 緊急の物資調達用に最低100円 |

<Section icon={Map} title="最終選別ステップバイステップ攻略">

**Demonfallの最終選別**を攻略するには、計画的なアプローチが必要です。森は広大で、道に迷うと複数の鬼が同時に出現する「危険地帯」に入り込んでしまうことがあります。以下のステップに従って、安全に鬼殺隊へと昇格しましょう。

<Steps>
  <Step num="1" title="選別のNPCを見つける">
    最終選別エリアの入り口に向かいます。藤の木の近くに2人のNPCが立っています。彼らに話しかけてレベル5の要件を満たしていることを確認し、テレポートを開始します。
  </Step>
  <Step num="2" title="安全圏を確保する">
    森に入ったら、中央へ突進してはいけません。出現パターンが予測しやすい端のほうにとどまりましょう。これにより、乱戦を避けて鬼を1体ずつおびき寄せて戦うことができます。
  </Step>
  <Step num="3" title="30体討伐の進め方">
    M1-M1-M1-M2コンボを使って鬼と交戦します。コンボが終わったら、必ずすぐにガード（Fキー）を押し続けてください。クリアフェーズに移行するには、30体の討伐を確定させる必要があります。
  </Step>
  <Step num="4" title="リソースの管理">
    10体倒すごとに、安全な木や岩の影に退避してスープを摂取しましょう。体力が低い場合は、強力な鬼の攻撃による出血効果を止めるために包帯を使用してください。
  </Step>
  <Step num="5" title="ゲートから脱出する">
    カウンターが30に達すると、通知が表示されます。光るランタンが目印のスタート地点のゲートに戻り、試練から脱出して報酬を受け取ります。
  </Step>
</Steps>

<Callout type="success" title="攻略ストラテジー">
もし多くの鬼に追われてしまった場合は、藤の木の方へ鬼をおびき寄せましょう。鬼は藤の木の境界に近づきすぎると継続的なダメージを受け、最終的に消滅するため、一息つくことができます。
</Callout>

</Section>

<Section icon={Swords} title="敵の種類と戦闘戦術">

**Demonfallの最終選別**に出現する鬼は、強さや能力がそれぞれ異なります。その多くは「通常の鬼（Generic Demons）」ですが、基本の血鬼術（Blood Demon Arts）や強力な突進攻撃を使用する俊敏な個体に遭遇することもあります。彼らの行動パターンを理解することが、勝利をつかむか、村へと引き返すはめになるかの分かれ道となります。

| 鬼の種類 | 危険度 | 戦闘戦略 |
| :--- | :--- | :--- |
| **弱い鬼** | 低 | 標準的なM1コンボ。ハメ（スタンロック）が容易。 |
| **素早い鬼** | 中 | 突進を待ち、左に回避してからカウンターを当てる。 |
| **強力な鬼** | 高 | 絶対に相打ちを狙わない。相手のM2をガードしてから反撃する。 |
| **青い鬼** | 極めて高 | レアスパン。ノックバック攻撃で距離を保つ。 |

<CardGrid cols={2}>
  <Card title="防御の立ち回り">
    - **ジャストガード（Perfect Block）**: 攻撃が当たる瞬間にガードを入力し、鬼を気絶（スタン）させます。
    - **ダッシュ回避（Dash Recovery）**: ガードゲージが壊れそうな場合は、Qキーを使ってダッシュで離脱します。
  </Card>
  <Card title="攻撃のコンボ">
    - **空中コンボ（Air Combo）**: M2で鬼を打ち上げ、追撃して追加ダメージを与えます。
    - **壁ハメ（Wall Pinning）**: 鬼を木に押し当ててハメ、ダッシュでの逃走を防ぎます。
  </Card>
</CardGrid>

<Callout type="tip" title="戦闘のコツ">
常にスタミナゲージ（青色のゲージ）に注意を払いましょう。スタミナが切れるとガードもダッシュもできなくなり、鬼のトドメの一撃に対して完全に無防備になってしまいます。
</Callout>
