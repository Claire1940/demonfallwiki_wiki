人間ビルドとハイブリッドビルドを比較すると、なぜソロ攻略において後者が好まれるのかがよく分かります。人間には隊士専用の強力なバフ（パーク）がありますが、長期戦となるレイドにおいては、ハイブリッドの圧倒的な生存力がダメージバフを上回ります。

| 項目 | 人間の隊士 | ハイブリッド |

<Section icon={Swords} title="ボス攻略：上弦の参・猗窩座（あかざ）">

**無限城**における最初の大きな関門は猗窩座です。彼の部屋に入ると、彼は即座に冨岡義勇をターゲットにします。プレイヤーの主な目的は、義勇がメインタンク（盾役）としてタゲを取っている間に、できる限り多くのダメージを与えることです。

<Steps>
  <Step num="1" title="開戦と接近">
    猗窩座が義勇をターゲットするのを待ちます。猗窩座がNPCに対してM2（重攻撃）を繰り出したら、すかさず「**突き（Lunge）**」で突進し、M1（通常攻撃）の3〜4ヒットコンボを叩き込みましょう。
  </Step>
  <Step num="2" title="義勇の救出（ピール）">
    もし義勇がコンボに捕まってしまったら、「**ショルダーバッシュ（Shoulder Bash）**」やノックバック効果の高い呼吸技を使って猗窩座を怯ませましょう。これによりボスのAIがリセットされ、デコイである義勇をより長生きさせることができます。
  </Step>
  <Step num="3" title="攻撃パターンの見極め">
    猗窩座のパンチには2つのバリエーションがあります。「**突進パンチ（Dash Punch）**」は横ダッシュで回避します。「**テレポートパンチ（Teleport Punch）**」はガード可能です。彼が画面から消えたら、すかさずガードを固めてください。
  </Step>
  <Step num="4" title="ソロへの移行">
    義勇が倒された後は、ソロでのガチンコ勝負になります。猗窩座の連続攻撃（乱打）を誘発させましょう。最初の攻撃をガードし、乱打モーションが終わるのを待ってから、ショルダーバッシュからフルコンボで反撃します。
  </Step>
</Steps>

| 猗窩座の技 | 対策アクション | 危険度 |
| :--- | :--- | :--- |
| **乱打（Barrage）** | ガード後にショルダーバッシュ | ★★★★☆ |
| **突進パンチ（Dash Punch）** | 横ダッシュ / 回避 | ★★★☆☆ |
| **テレポートキック（Teleport Kick）** | ガード維持 | ★★★☆☆ |
| **術式展開（Destructive Death）** | 距離を取る / 回避技 | ★★★★★ |

<Callout type="success" title="回復のチャンス">
ハイブリッドの場合、義勇がダウンするまで待ちましょう。ダウンした彼を担いで次の部屋に連れて行くか、その場ですぐに吸収（Body Absorption）することで、体力を全回復した状態で童磨戦に挑むことができます。
</Callout>

</Section>

<Section icon={Shield} title="ボス攻略：上弦の弐・童磨（どうま）と氷の御子">

童磨は、**無限城**においてまた異なるタイプの脅威となります。彼の戦闘はエリア制圧と雑魚処理（モブ管理）が中心です。ここでは胡蝶しのぶがアシストしてくれます。彼女は童磨の分身に対処する上で非常に重要です。

童磨が氷の御子（分身）を召喚した際、貴重な奥義（アルティメット技）を分身に消費してはいけません。分身の相手はしのぶに任せ、自分は本体への攻撃に集中しましょう。童磨の氷柱や氷の滑走は極めて危険ですが、予備動作（予兆モーション）が分かりやすいため、技の出かかりをキャンセルすることが可能です。

**童磨戦の戦術チェックリスト：**
- **2段ジャンプ**: 地面が凍りつくのが見えたら、氷の蔓（Ice Vines）を避けるためにいつでも2段ジャンプできるように構えておきましょう。
- **「突き」の連打**: 童磨が氷柱（Ice Pillar）の召喚モーションに入ったら、すかさず「突き（Lunge）」を放って技をキャンセルさせます。
- **回避技の温存**: 呼吸法の回避技は、童磨の掴み技や、複数の氷の滑走に巻き込まれた時のために温存しておきましょう。

| 童磨のスキル | 予兆（視覚的合図） | 最適な対処法 |
| :--- | :--- | :--- |
| **氷柱（Ice Pillars）** | 両腕を上に掲げる | **突き（Lunge）**でキャンセル |
| **氷の蔓（Ice Vines）** | 地面が白くなる | **2段ジャンプ** |
| **氷の御子（Ice Clones）** | 後方に跳躍する | しのぶにタゲを取らせる |
| **氷の滑走（Ice Slide）** | 前傾姿勢になる | ガードまたは横ダッシュ |

<Callout type="info" title="ドロップ情報">
童磨を倒すと、低確率で「**童磨の羽織（Doma's Cape）**」をドロップします。このアクセサリーは、2026年のアップデートにおいてその美しい見た目と僅かな防御バフから非常に人気の高いアイテムとなっています。
</Callout>

</Section>

<Section icon={Sparkles} title="最終決戦：上弦の壱・黒死牟（こくしぼう）と六つの目">

**無限城**のラスボスは、上弦の壱・黒死牟です。この戦いは忍耐力が試されます。不死川実弥が戦闘をサポートしてくれますが、黒死牟の広範囲に及ぶ範囲攻撃（AOE）により、実弥はすぐに倒されてしまうことが多いです。

**フェーズ1：サポート活用戦略**
実弥が生きている間は、「削り（ちくちくダメージを与えること）」に集中しましょう。黒死牟の「月の呼吸」は攻撃範囲が非常に広く、背後に回っていても被弾する可能性があるため、欲張って長いコンボを叩き込もうとしないことが重要です。中距離を保ち、飛び道具（遠距離攻撃）系の呼吸技を活用しましょう。

**フェーズ2：ソロでの生存戦略**
実弥が倒れた後は、ボスの攻撃を誘って後隙を叩く「待ち」の戦いになります。
- **遠距離戦**: 常にガードを維持してください。彼の三日月の斬撃はエリアの端から端まで届きます。
- **近距離戦**: 彼の周囲を円を描くように回り、連続攻撃を空振りさせましょう。攻撃を空振りさせたら、約2秒間の隙が生まれるので、そこでコンボを叩き込みます。

| 報酬 | ドロップ確率 | 効果 |
| :--- | :--- | :--- |
| **黒死牟の血（Kokushibo's Blood）** | 低確率（レア） | **六つの目（Six Eyes）**のバフを付与（鬼/ハイブリッド専用） |
| **闇の鉱石（Dark Ore）** | 高確率（コモン） | クラフト（製作）素材 |
| **プレステージXP** | 確定ドロップ | 大幅な育成ブースト |

<Callout type="warning" title="即死級ギミック">
黒死牟の最大出力の「月の呼吸」は、スタミナが切れた状態で受けると、ほとんどのプレイヤーが即死（ワンパン）します。この戦闘中は、スタミナゲージを絶対に20%未満に落とさないよう管理してください。
</Callout>

</Section>
