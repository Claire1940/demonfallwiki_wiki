<Section icon={Swords} title="Demonfall 戦闘の基本：核となるメカニクスと操作">

**Demonfallの戦闘**に従事するには、正確なタイミング、素早い反射神経、そして間合いの理解が必要です。標準的なアクションRPGとは異なり、ボタン連打はすぐに敗北に繋がります。すべての攻撃、ダッシュ、ブロックは、相手に硬直を突かれないよう計算されていなければなりません。

生存の基礎は、基本操作の習得にあります。以下は、PvEおよびPvPの両方の遭遇戦を切り抜けるために必要な必須入力の包括的な内訳です。

| アクション | キー設定 | 戦闘での活用法 |
| :--- | :--- | :--- |
| **弱攻撃** | 左クリック | 素早い近接攻撃でコンボを構築し、敵の体勢を削る |
| **強攻撃** | 右クリック | 動作は遅いがダメージが高く、ガードを崩したり敵を吹き飛ばしたりする |
| **ブロック** | F | 受けるダメージを軽減するが、強力なガードブレイク攻撃には弱い |
| **ダッシュ** | Q | 素早い方向移動で攻撃を回避したり、距離を詰めたりする |
| **呼吸** | G | 呼吸ゲージをチャージする。呼吸の技を繰り出すために不可欠 |

スタミナは防御プレイの生命線です。ブロックやダッシュのたびにスタミナを消費します。相手の強攻撃をブロックしている最中にスタミナバーが完全に空になると、体勢崩し（ガードブレイク）が発生し、スタン状態となってフルコンボを受ける隙を晒してしまいます。

体勢崩しを防ぐには、パーフェクトブロックを練習する必要があります。敵の攻撃が当たる直前にブロックキーを押すとパリィが発動し、攻撃者をスタンさせ、反撃を開始するための短いチャンスを得ることができます。

<Callout type="warning" title="ガードブレイクの危険性">
強攻撃（右クリック）は、相手のガードを粉砕するように設計されています。敵が強攻撃を溜めているのを見たら、ブロックしようとしてはいけません。代わりにダッシュ（Q）を使って横に移動し、敵の攻撃後の隙を攻撃しましょう。
</Callout>

</Section>

どのパスが自分の好みのプレイスタイルに合うかを判断するために、各勢力の主要な戦闘指標を比較してみましょう。

| 勢力 | 核心的なメカニクス | 戦闘の役割 | 難易度 |
| :--- | :--- | :--- | :--- |
| **人間** | 呼吸法と剣術 | バランスの取れたアタッカー / デュエリスト | 普通 |
| **鬼** | 血鬼術と体力再生 | 攻撃的なブローラー / サステイン | 普通 |
| **ハイブリッド** | 呼吸法と鬼のスキルの混合 | 高ダメージスペシャリスト | 難しい |

人間は戦場をコントロールするために、属性を伴う呼吸法に大きく依存します。常に呼吸ゲージとスタミナバーのバランスを取らなければなりません。一方、鬼は剣を使いません。生の肉体的な力と超自然的な血鬼術に頼ります。鬼は暗闇では急速に体力を回復しますが、日光や日輪刀からは甚大なダメージを受けます。ハイブリッドは、鬼の回復能力の一部を維持しつつ呼吸法を使用できる、究極のハイリスク・ハイリターンの戦闘体験を提供します。

</Callout>

<Section icon={Trophy} title="呼吸法ティアリスト：SティアからBティアまで">

人間およびハイブリッドのプレイヤーにとって、呼吸法は攻撃手段を定義する要素です。これらのスタイルは、基本攻撃のコンボに組み込める独自の型（スキル）を付与します。

| 呼吸法 | ティア | 戦闘の焦点 | 主要な技 |
| :--- | :--- | :--- | :--- |
| **日の呼吸** | Sティア | 高い炎上ダメージと長いコンボ | ヒノカミ神楽 |
| **月の呼吸** | Sティア | 広大な範囲攻撃と射程 | 厄鏡・月映え |
| **水の呼吸** | Aティア | 防御的なカウンターと流れるようなコンボ | 生生流転 |
| **霞の呼吸** | Aティア | 高い機動力と視覚的な撹乱 | 朧 |
| **雷の呼吸** | Aティア | 素早いダッシュ攻撃とバーストダメージ | 稲魂 |
| **風の呼吸** | Bティア | クラウドコントロールとノックバック | 爪々・風切 |
| **炎の呼吸** | Bティア | 高い素ダメージだが予測しやすいパターン | 煉獄 |

### Sティア
日の呼吸と月の呼吸は剣術の頂点を象徴しています。日の呼吸は、敵の自然回復を妨げる持続的な炎上状態を付与するため、長引くPvPデュエルで非常に強力です。月の呼吸は、広範囲に及ぶ三日月状の斬撃で複数の敵を寄せ付けず、広い半径で高いダメージを与えることに長けています。

### Aティア
水の呼吸、霞の呼吸、雷の呼吸は、ほとんどの戦闘シナリオで非常に優れたパフォーマンスを発揮する競争力の高い選択肢です。水の呼吸は最もバランスが取れており、敵のコンボをリセットする優れた防御的カウンター技を持っています。霞の呼吸は分身やテレポートのメカニクスを利用して相手を混乱させ、雷の呼吸は逃げるターゲットを捕らえるための瞬間的な接近に特化しています。

### Bティア
風の呼吸と炎の呼吸は堅実な選択肢ですが、予備動作が予測しやすいという欠点があります。風の呼吸はPvEでのクラウドコントロールに優れており、敵を集めて吹き飛ばすのに適しています。炎の呼吸は高いバーストダメージを叩き出しますが、経験豊富なPvPプレイヤーには予備動作の大きい技を簡単に見切られ、パリィされる可能性があります。

<Callout type="success" title="初心者へのおすすめ">
ゲームを始めたばかりの方は、水の呼吸から始めるのが良いでしょう。防御技の許容範囲が広く、序盤の修行クエストも比較的簡単にクリアできます。
</Callout>

</Section>

<Section icon={Target} title="ステップバイステップの進行：戦闘キャラクターのビルド方法">

強力なキャラクターを構築するには、レベリング、スキルポイントの割り振り、装備の獲得に対する構造的なアプローチが必要です。以下の手順に従って、戦闘ビルドを効率的に最適化しましょう。

<Steps>
  <Step num="1" title="勢力と一族の選択">
    キャラクターを作成し、一族（クラン）のロールを行います。上位の一族は、体力やスタミナの増加、あるいは基礎戦闘能力を高める独自の家系の技など、パッシブなステータスボーナスを提供します。
  </Step>
  <Step num="2" title="早川村での初心者クエスト">
    早川村周辺で初期の配達や狩猟クエストを完了し、基本的な経験値と円を確保します。この通貨を使って最初の本格的な武器を購入したり、基本スキルを解放したりします。
  </Step>
  <Step num="3" title="育手を見つけてスタイルを解放">
    マップを移動して、選んだスタイルの育手（トレーナー）を見つけます。肉体トレーニングや敵の撃破など、特定の試練をクリアして、最初の戦闘スキルセットを解放しましょう。
  </Step>
  <Step num="4" title="ボスを倒して素材を収集">
    他のプレイヤーと協力して主要なボスを狩ります。ドロップした素材を使用して高度な日輪刀を鍛造したり、「呼吸の起訴状（Breath Indictment）」のようなレアな進行アイテムを入手したりします。
  </Step>
  <Step num="5" title="プレステージでレベル上限を解放">
    最大レベルに達したら、プレステージNPCに話しかけてレベルを1にリセットします。このプロセスにより最大レベル上限が上がり、恒久的なスキルポイントが付与され、全体的なパワーが底上げされます。
  </Step>
</Steps>

<Callout type="tip" title="プレステージの利点">
プレステージを行うとレベルとストーリーの進行状況はリセットされますが、一族、呼吸法、解放したコスメティックアイテムは維持されます。プレステージで得られる追加のスキルポイントは、競争力のあるエンドゲームビルドを作成するために不可欠です。
</Callout>

</Section>

<Section icon={Sparkles} title="エンドゲームの挑戦：ボス戦の戦略">

エンドゲームのボスを倒すことは、レアなクラフト素材、円、そして膨大な経験値を獲得するための主要な方法です。しかし、ボスの遭遇戦には、準備不足のプレイヤーを簡単に全滅させる独自のメカニクスが備わっています。

| ボス | 場所 | 戦闘戦略 | 推奨レベル |
| :--- | :--- | :--- | :--- |
| **累** | 蜘蛛山 | 糸のトラップを回避し、硬直中に攻撃する | レベル 25+ |
| **猗窩座** | 中央の荒野 | 衝撃波攻撃を避けるために動き続け、突きをパリィする | レベル 40+ |
| **妓夫太郎** | 遊郭 | 毒耐性を準備し、大鎌のなぎ払い攻撃を回避する | レベル 45+ |
| **獪岳** | 無限城 | 落雷を避けるために距離を取り、詠唱を妨害する | レベル 50+ |

### 累の戦略
累は、プレイヤーを拘束する遠距離の糸攻撃と、その後の高ダメージの叩きつけに依存しています。彼を倒すには、中距離を維持し、彼が糸を引く動作をするのを待ちます。アニメーションが始まったらすぐに横にダッシュし、距離を詰めてフルコンボの弱攻撃を叩き込みましょう。

### 猗窩座の戦略
猗窩座は、素早い拳の連打と爆発的な衝撃波を駆使する超攻撃的なボスです。彼のコンボをすべてブロックしようとすると、すぐに体勢を崩されてしまいます。代わりに、初撃をパリィして彼の流れを断ち切るか、機動力の高い呼吸法の技を使って距離を取りつつ体力を削ることに集中しましょう。
