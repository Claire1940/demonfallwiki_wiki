<Section icon={Target} title="Como Resgatar Códigos no Demonfall">

Ao contrário de muitos jogos do Roblox que usam um botão dedicado na interface (GUI) para recompensas, este jogo utiliza o sistema de chat. Isso pode ser confuso para iniciantes que procuram por um menu de "Códigos". Siga o protocolo específico abaixo para garantir que seus **códigos de demonfall** sejam aceitos pelo servidor.

<Steps>
  <Step num="1" title="Abra a Interface de Chat">
    Pressione a tecla "/" no seu teclado ou clique no ícone do balão de chat no canto superior esquerdo da tela para ativar o campo de entrada.
  </Step>
  <Step num="2" title="Insira o Comando">
    Digite o código exatamente como ele aparece, incluindo o prefixo "!". Por exemplo, digite `!update6` na caixa de chat.
  </Step>
  <Step num="3" title="Envie e Verifique">
    Pressione "Enter". Se for bem-sucedido, uma notificação aparecerá acima da sua barra de atalhos e os itens serão adicionados diretamente ao seu inventário.
  </Step>
</Steps>

Se um código não funcionar, geralmente é devido a um dos problemas comuns listados na tabela abaixo.

| Mensagem de Erro / Problema | Causa Provável | Solução Recomendada |
| :--- | :--- | :--- |
| **Invalid Code** | Erro de digitação ou falta do prefixo "!" | Digite novamente com cuidado, garantindo que não haja espaços extras. |
| **Code Expired** | O código atingiu seu limite de tempo | Passe para o próximo código ativo. |
| **Already Redeemed** | Você já usou este código nesta conta | Os códigos geralmente são de uso único por personagem. |
| **Server Lag** | Alto tráfego após uma atualização | Aguarde 60 segundos e tente novamente em um servidor privado. |

<Callout type="warning" title="Aviso Importante">
Itens como Wipe Potions resetarão todo o progresso do seu personagem, incluindo seu nível e clã. Não use esses códigos a menos que tenha certeza de que deseja um novo começo.
</Callout>

</Section>
